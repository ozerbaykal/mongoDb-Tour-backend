# Tour Project Backend

This is the backend for the Tour Project, built using Node.js and Express. It utilizes MongoDB as the database and includes various features such as authentication, authorization, and tour management.

## Technologies Used

- Database: MongoDB

- Framework: Express.js

- Authentication & Authorization: JSON Web Token (JWT), bcrypt

- Security: Helmet, Express-rate-limit, Express-mongo-sanitize, HPP

- File Uploads: Multer, Sharp

- Validation: Validator

- Environment Variables: Dotenv

## Controllers

The project includes the following controllers:

- reviewController: Handles user reviews, including adding and deleting reviews.

- userController: Manages user-related actions.

- tourController: Manages tour-related actions, such as retrieving, updating, and deleting tour details.

- authController: Handles authentication and authorization processes.

## Features

- Users can create, update, and delete tours.

- Users can post, edit, and delete reviews.

- Authentication & Authorization using JWT.

- Middleware for security enhancements.
